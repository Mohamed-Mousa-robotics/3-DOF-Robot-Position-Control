# 3-DOF-Robot-Position-Control
PID plus Inverse Dynamic for position and velocity control of 3 DOF robot
# Overview 
The dynamic model of the 3 DOF robot was derived manually using newton Euler method. The inertia coupling was neglected to reduce the complexity of the model.  
# Control Strategy
The integration of inverse dynamic and PID control methods was donr in order to control the position and velocity of the 3 DOF articulated robot.
